Binary (base - 2)
	This is how computers today count
	They count in a term called binary digit also known as a **bit** 

Bit
	A zero or one: on or off
	This is how computers speak. A zero represents off and a one equals on.
	Each device that you use on the daily has millions or billions of transistors that are being turned on and off.

Byte
	8 bits turn into a byte
	00000000
	128 64 32 16 8 4 2 1
	256 possibilities when you take in account zero
	Some computers used to be able to only show 256 colors or etc.

ASCII (American Standard Code for Information Interchange)
	7 - bit character code where each individual bit represents a unique character.

Unicode
	Uses more bits than ASCII
	Like emojis
	Assigns a unique number to every character, symbol, and punctuation mark across all languages

RGB
